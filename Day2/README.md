##task_2_4.py 
Вводим строку а затем выводим эту же строку без послдних 2-х символов.
Делаем с помощью среза. Находим длинну строки, от длинны строки отнимаем 2 символа, и выводим срез от 0 до длинна строки -2

##task_2_5.py 
Вводим строку а затем создаем новую из четных элементов строки. 
В цикле проходим по всей длинне строки, проверям остаток от деления индекса на 2, если 0 то записываем элемент строки в новую строку.
